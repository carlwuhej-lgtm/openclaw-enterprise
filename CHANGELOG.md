All notable changes to OpenClaw Enterprise will be documented in this file.

## [1.0.0] - 2026-03-30

### Added
- Initial release of OpenClaw Enterprise
- Core security governance features
- JWT authentication with role-based access control
- Audit logging with sensitive data filtering
- Policy engine for real-time risk interception
- Multi-tenant support
- Agent client for 5 platforms (Windows, macOS, Linux)
- Web-based management dashboard
- API documentation
- Network scanning for OpenClaw instance discovery
- Alert center with multi-channel notifications
- Analysis and reporting (PDF export)

### Security
- Password hashing with SHA256+Salt
- Rate limiting (60 requests/minute)
- XSS/CSRF protection
- Input validation
- Audit log sensitive field filtering

### Documentation
- README with quick start guide
- API reference documentation
- Deployment guide
- User manual
- Security policy
- Contributing guidelines
