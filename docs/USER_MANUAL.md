## 1. 产品概述

### 1.1 产品简介

OpenClaw Enterprise 是专为企业设计的 OpenClaw 集中管控平台，解决企业内部员工私自安装 OpenClaw 导致的数据泄露与违规使用风险。

### 1.2 核心价值

- 🔐 **统一管控** - 发现、注册、审批所有 OpenClaw 实例
- 🛡️ **安全合规** - 策略管控、行为审计、风险告警
- 🏢 **多租户隔离** - 部门/团队独立管理，权限分离

### 1.3 适用角色

| 角色 | 权限 | 适用人员 |
|------|------|----------|
| 系统管理员 | 全部权限 | IT 运维人员 |
| 安全管理员 | 策略、告警、审计权限 | 安全团队成员 |
| 普通用户 | 查看权限（本部门） | 部门主管、普通员工 |

## 2. 快速入门

### 2.1 系统登录

> **注意**: 当前版本为演示版本，无需登录即可访问。生产环境将启用认证系统。

1. 打开浏览器
2. 访问：`http://localhost:8000/pages/dashboard`
3. 进入系统主界面

### 2.2 界面导览

#### 侧边栏导航
- 位于界面左侧，固定显示
- 包含 6 个主菜单
- 点击菜单切换页面
- 当前页面对应菜单高亮显示

#### 顶部栏
- **面包屑**: 显示当前位置
- **搜索框**: 全局搜索设备、用户、告警
- **通知按钮**: 查看系统通知
- **用户信息**: 显示当前用户信息

#### 主内容区
- 显示当前页面的主要内容
- 支持卡片、表格、图表等多种展示形式

### 2.3 操作流程示例

**场景**: 发现并处理一台违规设备

1. **发现告警** → 控制台看到"未授权设备接入"告警
2. **查看详情** → 点击告警中心查看详细信息
3. **定位设备** → 点击设备链接跳转到资产管理
4. **阻断设备** → 点击"阻断"按钮，确认操作
5. **验证结果** → 返回控制台确认告警已消除

## 3. 控制台

### 3.1 功能概述

控制台是系统的首页，提供全局视图和快速操作入口。

### 3.2 统计卡片

控制台顶部显示 4 个统计卡片：

| 卡片 | 显示内容 | 说明 |
|------|----------|------|
| 💻 已注册实例 | 设备总数、本周新增 | 点击可跳转到资产列表 |
| ⏳ 待审批设备 | 等待审批的设备数 | 红色高亮，需及时处理 |
| 🚨 高危告警 | 未解决的严重告警数 | 点击跳转到告警中心 |
| ✅ 今日审计日志 | 今日操作日志总数 | 显示趋势变化 |

### 3.3 设备列表

**显示内容**:
- 设备名称和图标
- 所属用户和部门
- 运行状态（在线/离线/需升级/违规）
- 风险等级（低/中/高 + 风险条）
- 最后活跃时间
- 操作按钮（详情/阻断）

**快速操作**:
| 按钮 | 功能 | 说明 |
|------|------|------|
| 🔍 扫描网络 | 自动发现内网设备 | 系统自动扫描并列出新设备 |
| ➕ 手动注册 | 手动添加设备 | 填写设备信息后提交 |

### 3.4 实时审计

显示最近 5 条操作日志，支持按类型筛选：
- 全部
- 命令执行
- 文件访问
- 网络请求

### 3.5 实时告警

显示最近 4 条活跃告警：
- 🔴 严重告警（红色背景）
- 🟠 高危告警（橙色背景）
- 🔵 中危告警（蓝色背景）

### 3.6 安全策略

快速查看和切换 7 条核心策略的启用状态：
1. 禁止访问敏感目录
2. 强制密钥管理
3. 限制外部模型连接
4. 第三方技能沙箱
5. 高危命令需审批
6. 操作日志实时上报
7. 自动版本升级

**操作方法**:
1. 点击开关启用/禁用策略
2. 点击"保存配置"提交更改

### 3.7 租户分布

以进度条形式展示各部门的设备分布：
- 研发中心（蓝色）
- 运维部（绿色）
- 数据部（橙色）

## 4. 资产管理

### 4.1 功能概述

管理所有注册的 OpenClaw 实例，包括设备发现、注册、审批、阻断等操作。

### 4.2 设备列表

**列表信息**:
| 列名 | 说明 |
|------|------|
| 设备信息 | 设备名称、主机名、所属租户、版本 |
| 用户 | 设备使用人 |
| 部门 | 所属部门 |
| 版本 | OpenClaw 版本号 |
| 状态 | 在线/离线/需升级/违规 |
| 风险等级 | 低/中/高（带风险评分） |
| 最后活跃 | 最后在线时间 |
| 操作 | 详情/阻断按钮 |

### 4.3 设备发现

**自动扫描**:
1. 点击"扫描网络"按钮
2. 系统自动扫描内网中的 OpenClaw 实例
3. 扫描完成后显示新设备列表
4. 逐个或批量审批新设备

**手动注册**:
1. 点击"手动注册"按钮
2. 填写设备信息：
   - 设备名称（必填）
   - 主机名（必填）
   - 版本号（必填）
   - 所属租户（必填）
   - 使用用户（可选）
3. 点击"提交"完成注册

### 4.4 设备详情

点击"详情"按钮查看设备完整信息：

**基本信息**:
- 设备名称
- 主机名
- IP 地址
- MAC 地址
- OpenClaw 版本
- 操作系统信息

**归属信息**:
- 所属租户
- 使用用户
- 注册时间
- 最后活跃时间

**状态信息**:
- 运行状态
- 风险等级
- 风险评分（0-100）
- 风险因素分析

**操作记录**:
- 最近操作日志
- 告警历史
- 策略违规记录

### 4.5 设备阻断

**阻断违规设备**:
1. 找到状态为"违规"的设备
2. 点击"阻断"按钮
3. 确认阻断操作
4. 设备被加入黑名单，无法连接内网资源

**解除阻断**:
1. 进入设备详情
2. 点击"解除阻断"
3. 填写解除原因
4. 提交审核（需管理员审批）

### 4.6 设备编辑

**修改设备信息**:
1. 进入设备详情
2. 点击"编辑"按钮
3. 修改需要变更的信息
4. 点击"保存"提交

**可修改项**:
- 设备名称
- 所属租户
- 使用用户
- 备注信息

### 4.7 设备删除

**删除设备**:
1. 进入设备详情
2. 点击"删除"按钮
3. 确认删除（不可恢复）
4. 设备记录被永久删除

**注意**: 删除前请确保：
- 设备已离线
- 无未处理的告警
- 已备份相关数据

### 4.8 搜索与筛选

**搜索**:
- 在搜索框输入关键字
- 支持设备名、主机名、用户名搜索
- 实时过滤结果

**筛选**:
- 状态筛选：在线/离线/需升级/违规
- 风险筛选：低/中/高
- 租户筛选：按部门筛选
- 时间筛选：按注册时间筛选

### 4.9 批量操作

**操作步骤**:
1. 勾选需要操作的设备
2. 点击顶部"批量操作"按钮
3. 选择操作类型：
   - 批量审批
   - 批量阻断
   - 批量删除
4. 确认操作

## 5. 策略中心

### 5.1 功能概述

配置和管理企业安全策略，确保 OpenClaw 合规使用。

### 5.2 策略列表

系统预置 7 条核心安全策略：

#### 1. 🚫 禁止访问敏感目录

**功能**: 阻止 OpenClaw 访问系统敏感目录

**配置项**:
- 敏感目录列表
- 阻断动作（记录/告警/阻断）
- 例外目录（白名单）

**默认配置**:
```
/etc
/root
/credentials
~/.ssh
~/.aws
~/.kube
```

#### 2. 🔑 强制密钥管理

**功能**: 强制使用企业密钥管理系统

**配置项**:
- 密钥管理提供商
- 服务地址
- 认证方式

**支持提供商**:
- HashiCorp Vault
- 1Password
- AWS Secrets Manager
- 企业自建服务

#### 3. 🌐 限制外部模型连接

**功能**: 限制连接未经审批的 LLM 服务

**配置项**:
- 允许的模型列表
- 阻断动作
- 例外用户（白名单）

**默认允许**:
- internal-glm
- internal-qwen
- internal-llama

#### 4. 📦 第三方技能沙箱

**功能**: 第三方技能在隔离环境中运行

**配置项**:
- 沙箱类型
- 资源限制（CPU/内存）
- 网络访问权限

**沙箱类型**:
- Docker 容器
- Firecracker 微虚拟机
- 无隔离（不推荐）

#### 5. ⚠️ 高危命令需审批

**功能**: 执行高危命令需要管理员审批

**配置项**:
- 高危命令列表
- 审批人员
- 审批超时时间

**默认命令**:
```
rm -rf /
dd if=/dev/zero
chmod 777
mkfs
fdisk
```

#### 6. 📊 操作日志实时上报

**功能**: 所有操作日志实时上报到 SIEM

**配置项**:
- SIEM 端点地址
- 上报频率
- 日志格式

**支持 SIEM**:
- Splunk
- ELK Stack
- QRadar
- 企业自建 SIEM

#### 7. 🔄 自动版本升级

**功能**: 自动升级到最新安全版本

**配置项**:
- 自动升级开关
- 维护窗口时间
- 升级通知方式

**建议配置**:
- 启用自动升级
- 维护窗口：02:00-04:00
- 通知方式：邮件 + 飞书

### 5.3 策略操作

**启用/禁用策略**:
1. 找到目标策略
2. 点击右侧开关
3. 蓝色=启用，灰色=禁用
4. 点击"保存配置"生效

**编辑策略配置**:
1. 点击策略名称
2. 进入策略详情页
3. 修改配置参数
4. 点击"保存"提交

**策略继承**:
- 租户可继承全局策略
- 租户可覆盖部分策略
- 子租户继承父租户策略

### 5.4 策略说明

右侧显示策略详细说明：
- 策略功能
- 配置说明
- 最佳实践
- 注意事项

**时间范围预设**:
- 最近 1 小时

3. 实时显示匹配结果

**搜索示例**:
- 搜索文件路径：`main.go`

**导出内容**:
- 当前筛选条件下的所有日志

- 自动添加导出时间和操作人

### 6.7 日志统计

4. 查看图表

4. 确认操作

**告警详情**:

### 7.6 告警统计

## 8. 租户管理

### 8.1 功能概述

管理企业各部门/团队的 OpenClaw 租户，实现隔离和分权。

### 8.2 租户列表

**显示信息**:
| 列名 | 说明 |
|------|------|
| 租户信息 | 租户名称、描述 |
| 代码 | 租户唯一标识 |
| 设备数 | 该租户下设备总数 |
| 用户数 | 该租户下用户总数 |
| 状态 | 活跃/禁用 |
| 创建时间 | 租户创建日期 |
| 操作 | 编辑/设备/删除 |

### 8.3 新建租户

**操作步骤**:
1. 点击"新建租户"按钮
2. 填写租户信息：
   - 租户名称（必填）
   - 租户代码（必填，唯一）

## 10. 常见问题

### 10.1 登录相关

**Q: 忘记密码怎么办？**
A: 点击登录页"忘记密码"，通过注册邮箱重置密码。

**Q: 账号被锁定怎么办？**
A: 联系系统管理员解锁账号。

### 10.2 设备管理

**Q: 设备扫描不到怎么办？**
A: 
1. 确认设备已安装 OpenClaw
2. 确认设备与服务器网络连通
3. 确认设备防火墙未阻挡扫描

**Q: 设备阻断后如何恢复？**
A: 
1. 进入设备详情
2. 点击"解除阻断"
3. 填写解除原因
4. 管理员审批后恢复

### 10.3 策略配置

**Q: 策略修改后多久生效？**
A: 策略保存后立即生效，设备下次心跳时同步策略。

**Q: 如何给特定租户设置独立策略？**
A: 
1. 进入租户管理
2. 选择目标租户
3. 点击"策略"标签
4. 配置租户独立策略

### 10.4 审计日志

**Q: 日志最多保留多久？**
A: 默认保留 180 天，可在系统设置中调整。

**Q: 导出的日志文件在哪里？**
A: 默认下载到浏览器指定的下载目录。

### 10.5 告警处理

**Q: 告警太多处理不过来怎么办？**
A: 
1. 优先处理严重和高危告警
2. 使用批量解决功能
3. 配置告警自动解决规则

**Q: 如何避免告警风暴？**
A: 
1. 配置告警聚合规则
2. 设置告警静默时间
3. 调整告警阈值

### 10.6 系统性能

**Q: 系统响应慢怎么办？**
A: 
1. 检查服务器资源使用情况
2. 清理过期日志
3. 优化数据库索引

**Q: 支持多少设备并发？**
A: 单实例支持 1000+ 设备并发，大规模部署建议使用集群模式。
